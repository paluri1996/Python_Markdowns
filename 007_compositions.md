# PYTHON Convert Tuple to List to Dictionary
```PYTHON
#convert 2 Lists to Dict
print(dict(zip([x for x in 'abcdefg'], range(1, 8))))
OUTPUT: {'a': 1, 'b': 2, 'c': 3, 'd': 4, 'e': 5, 'f': 6, 'g': 7}
```
